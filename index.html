🔥 Conectar con Firebase

Esta app usa Google Firebase para sincronizar los datos entre todas las computadoras de la fábrica en tiempo real.
Solo necesitas hacerlo una vez por equipo. La configuración se guarda localmente.

Cómo obtener los datos:
1. Ve a console.firebase.google.com
2. Crea un proyecto (ej: foga-equifrigo)
3. Agrega una app Web (</>)
4. En Firestore Database → crea la base en modo producción
5. En Configuración del proyecto → copia los valores de abajo
6. En Firestore → Reglas → pega:
allow read, write: if true;
⚠️ Verifica los datos e intenta de nuevo.
